A Simplified Explanation of Linear Search

Written By Ajinkya Dhatrak March 31, 2023
Searching

Detailed understanding of the working of the linear search algorithm and its implementation in C++

A Linear Search is the most basic type of searching algorithm. A Linear Search sequentially moves through your collection (or data structure) looking for a matching value. In other words, it looks down a list, one item at a time, without jumping.

Think of it as a way of finding your way in a phonebook. A Linear Search is starting at the beginning, reading every name until you find what you're looking for.

Problem Statement

Given an array arr[] of n elements, write a function to search a given element x in arr[] and return the index of x in the array.

Examples

Input : arr[] = {10, 20, 30, 50, 60, 80, 110, 130, 140, 170}, x = 110
Output : 6
Explanation : Element x is present at index 6.

Input : arr[] = {10, 20, 30, 40, 60, 110, 120, 130, 170}, x = 175
Output : -1
Explanation : Element x is not present in arr[].

Algorithm

  1. First, read the search element (Target element) in the array.
  2. Set an integer i = 0 and repeat steps 3 to 4 till i reaches the end of the array.
  3. Match the key with arr[i].
  4. If the key matches, return the index. Otherwise, increment i by 1.

Code


    #include <iostream>
    using namespace std;

    int search(int arr[], int n, int x)
    {
        int i;
        for (i = 0; i < n; i++)
            if (arr[i] == x)
                return i;
        return -1;
    }
    
    // Driver code
    int main(void)
    {
        int arr[] = { 2, 3, 4, 10, 40 };
        int x = 10;
        int n = sizeof(arr) / sizeof(arr[0]);
    
        // Function call
        int result = search(arr, n, x);
        (result == -1) ? cout << "Element is not present in array" 
        : cout << "Element is present at index " << result;
        return 0;
    }

Output


    Element is present at index 3

Time & Space Complexity

Time complexity: O(N)
Auxiliary Space: O(1)
